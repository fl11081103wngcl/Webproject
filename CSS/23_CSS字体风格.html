骑士真的很受伤，不知道从什么时候开始，世间的公主都爱恶龙，都向往和恶龙在一起。就连他暗恋了八年的公主，也总和他吐露希望哪天能被恶龙掳走。 既然如此，那也只能这样了。骑士披上了龙皮，在一个夜黑风高的晚上，掳走了心爱的公主。 “恶龙”如愿以偿和公主在一起了，在滚床单之前，“恶龙”假意让公主吻他一下，说，“你发自真心的吻，会让我显出人形，以你真心爱的形象。”然后让龙皮脱落，现出人形。 公主认出了骑士，从此，骑士结束了跑龙套的生活。

恶龙们历代守护公主，不作恶，不闹事，每次都将公主照顾得很好，还能在俊美的骑士到来之时，假装被打倒，让公主和帅气的骑士在一起。 它们终于得到了全人类的认可，不再以当公主的护卫为主业，而且成为国家队，保护国家安全。 “这就是我为什么站在这里的原因。”恶龙向路边闯红灯的小骑士解释道。 “所以，请叫我龙井茶。”
