第一步：安装插件的包管理器，package control 安装上
1.ctrl+~ ，打开命令执行窗口
2.然后把安装包管理器的代码黏贴到命令执行窗口
3.重启Sublime
命令代码：sublime3安装命令行： import urllib.request,os,hashlib; h = 'eb2297e1a458f27d836c04bb0cbaf282' + 'd0e7a3098092775ccb37ca9d6b2e4b7d'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)
第二步：第三方插件安装 1.ctrl+shit+p 打开命令面板
2.然后输入install 回车
3.输入包的关键字进行搜索，回车安装
